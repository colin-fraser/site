In 2020, we're doing our own websites

2020-01-19

I’ve written a lot in other places—mostly Medium, but I think in 2020 we’re doing our own websites. Blogdown has gotten so good that it feels like a big waste of time to write posts in RMarkdown and translate them over to Medium like I’ve been doing. So this is where I’m going to write my stuff now. I’ll probably spend some time migrating some of the existing posts over to this site too.
